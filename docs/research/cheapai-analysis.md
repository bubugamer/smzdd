# GetCheapAI.com 调研分析

**调研日期**: 2026-02-07
**网站**: https://www.getcheapai.com/zh-cn
**定位**: 以模型为中心的AI模型价格对比平台

## 网站特点

### 核心功能
1. **模型中心视图**: 2,600+个AI模型的价格对比
2. **多中转站对比**: 显示每个模型在不同中转站的价格
3. **历史价格趋势**: 价格变化图表和时间线
4. **多货币支持**: CNY, USD, JPY, EUR, KRW, TWD, HKD
5. **计费类型筛选**: 按token计费 vs 按请求计费

### 数据展示方式
- 模型列表（热门206个 + 全部2,600+）
- 价格对比表格（输入/输出价格）
- 支持中转站数量统计
- 最后更新时间戳
- 折扣信息和优惠活动

### 与HelpAIO的区别

| 维度 | GetCheapAI | HelpAIO |
|------|-----------|---------|
| 视角 | 模型中心 | 服务商中心 |
| 数据量 | 2,600+模型 | 17家中转站 |
| 核心功能 | 价格对比 | 综合评分+可用性 |
| 货币 | 多货币 | 仅人民币 |
| 历史数据 | 价格趋势图 | 价格变动时间线 |

## 对SMZDD项目的启示

### 1. 双视角设计
- **服务商视角**（类似HelpAIO）：适合选择中转站
- **模型视角**（类似GetCheapAI）：适合比较特定模型价格
- 建议：初期以服务商视角为主，后期增加模型视角

### 2. 数据结构设计
```
Provider (服务商)
  ├── ProviderModel (关联)
  │   ├── Model (模型)
  │   └── Pricing (定价)
  └── Reviews (评价)
```

这种结构支持两种视角：
- 从Provider查询所有Model
- 从Model查询所有Provider

### 3. 功能规划

**阶段1（MVP）**:
- 服务商列表和详情
- 基本价格对比
- 手动数据录入

**阶段2（增强）**:
- 模型中心视图
- 高级筛选
- 价格趋势图

**阶段3（完善）**:
- 多货币支持
- 历史价格分析
- 自动化数据更新

### 4. 数据收集策略

**手动收集**（初期）:
1. 访问GetCheapAI网站
2. 选择热门模型（如Claude Sonnet, GPT-4等）
3. 记录各中转站价格
4. 整理为CSV格式

**自动化收集**（后期）:
1. 开发网页爬虫
2. 定期抓取价格数据
3. 对比检测价格变动
4. 自动更新数据库

## 关键发现

### 1. 价格差异显著
- 同一模型在不同中转站价格差异可达50-100%
- 价格竞争激烈
- 用户对价格敏感

### 2. 模型多样性
- 2,600+模型说明市场需求多样
- 但实际热门模型集中在前50个
- 长尾模型关注度低

### 3. 更新频率
- 价格变动频繁
- 需要定期更新机制
- 最后更新时间很重要

### 4. 用户需求
- 快速找到最便宜的中转站
- 了解价格历史趋势
- 获取优惠信息
