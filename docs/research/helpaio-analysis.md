# HelpAIO.com 调研分析

**调研日期**: 2026-02-07
**网站**: https://www.helpaio.com/transit
**定位**: 以服务商为中心的AI API中转站对比平台

## 网站特点

### 核心功能
1. **综合评分系统**: 评分 = 基础分 × 可用率
2. **实时可用性监控**: 3天可用率统计
3. **服务商排名**: 基于综合评分排序
4. **详细服务商档案**: 包含优缺点、定价、支付方式
5. **多维度筛选**: 按模型类型、定价模式、评分排序

### 数据展示方式
- 网格/列表视图切换
- 实时可用性徽章
- 价格变动时间线
- 用户评价和反馈

## 关键发现

### 1. 评分体系
- 综合评分 = 基础分 × 可用率
- 基础分考虑：定价、功能、支持质量
- 可用率基于3天实时监控

### 2. 价格竞争
- 价格差异显著（最高vs最低约76%）
- 性价比成为关键竞争点
- 包月套餐适合重度用户

### 3. 稳定性
- 头部中转站可用率普遍>99%
- 稳定性是核心竞争力
- 缓存命中率影响用户体验

### 4. 支付方式
- 国内支付为主（支付宝、微信）
- 部分支持银行卡
- 缺少国际支付方式

## 对SMZDD项目的启示

### 1. 数据模型设计
- 需要存储综合评分和基础分
- 可用率需要实时计算（基于探针数据）
- 价格历史跟踪很重要

### 2. 功能优先级
- 实时可用性监控（核心功能）
- 价格对比（用户最关心）
- 评分系统（帮助决策）

### 3. 数据收集策略
- 初期手动收集17家中转站数据

- 定期更新价格和可用性

### 4. 监控指标
- 3天可用率
- 响应时间
- 价格变动
- 缓存命中率（如果能获取）
